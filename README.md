📖 Présentation du projet
Le Système de Gestion des Auditoires (SGA) est une application web développée en PHP procédural permettant d’automatiser la gestion des auditoires et la planification des horaires des cours au sein de l’Université Protestante au Congo.
Le système facilite :


La gestion des salles et auditoires


L’attribution automatique des horaires


La gestion des promotions et options


La planification hebdomadaire des cours


La gestion des utilisateurs selon leurs rôles


Le suivi des disponibilités des salles


Ce projet vise à réduire les conflits d’horaires, améliorer l’organisation académique et moderniser la gestion administrative des auditoires.

🎯 Objectifs du projet
Objectif général
Mettre en place une plateforme web capable de gérer efficacement les auditoires et les horaires universitaires.
Objectifs spécifiques


Automatiser la réservation des auditoires


Éviter les chevauchements des cours


Optimiser l’utilisation des salles


Faciliter la consultation des horaires


Sécuriser l’accès selon les rôles utilisateurs

🚀 Installation du projet
Prérequis


PHP 8+


XAMPP ou WAMP


Navigateur web


Étapes d’installation
1. Cloner ou copier le projet
git clone projet_sga
2. Placer le dossier dans htdocs
C:\xampp\htdocs\
3. Démarrer Apache avec XAMPP
4. Ouvrir le navigateur
http://localhost/gestion_audi
